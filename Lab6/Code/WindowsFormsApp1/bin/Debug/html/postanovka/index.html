Постановка задачи

В данной работе рассматриваются объявление и инициализация вещественной константы в СУБД PostgreSQL. В общем виде
перечисления выглядят так:
ИДЕНТИФИКАТОР constant numeric := ЧИСЛО ;
Где идентификатор – это название переменной, а число – одно вещественное число. Примеры правильных перечислений:

  • vat constant numeric := 4.5 ;
  • anything constant numeric := 1.125 ;
