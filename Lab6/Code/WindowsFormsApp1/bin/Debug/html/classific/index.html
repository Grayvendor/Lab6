Классификация грамматики

Согласно классификации Хомского, полученная порождающая грамматика G[] соответствует типу
контекстно-свободных, так как правая часть каждой редукции начинается либо с терминального символа, либо с
нетерминального, принадлежащего объединённому словарю.

Грамматика G[] не является автоматной, так как не все её редукции начинаются с терминального
символа. По этой же причине данная грамматика не является S - грамматикой.
