Панель инструментов

Панель инструментов (см. рисунок А.7) содержит кнопки вызова часто используемых пунктов меню:

  1. Создание документа
  2. Открытие документа
  3. Сохранение текущих изменений в документе
  4. Отмена изменений
  5. Повтор последнего изменения
  6. Вырезать текстовый фрагмент
  7. Вырезать текстовый фрагмент
  8. Копировать текстовый фрагмент
  9. Вставить текстовый фрагмент
  10. Запуск синтаксического анализатора

Пункт меню "Файл"

  1. Создать - создание файла
  2. Открыть - открытие файла
  3. Сохранить - сохранение текущего документа
  4. Сохранить как - сохранине файла с указанием места
  5. Выход - закрывает программу

Пункт меню "Правка"

  1. Отменить - отменяет последние изменения
  2. Повторить - возвращает отмененные изменения
  3. Вырезать - вырезает выделенный фрагмент текста
  4. Копировать - копирует выделенный текст
  5. Вставить - вставляет текст из буфера обмена в документ
  6. Удалить - удаляет выделенный фрагмент текста
  7. Выделить все - выделяет все в документе
